Popular Movies App
==================

A simple iOS app, that helps user to discover movies. 
This is the copy of android app for Project 1 & Project 2 of Udacity's Android Developer Nanodegree.

**Features:**

- Discover the most popular, the highest rated and the most rated movies
- Watch movie trailers and teasers
- Read reviews from other users
- Mark movies as favorites
- Search for movies
- Offline work
- Material design
- UI optimized for phone and tablet
